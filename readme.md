
# Pomodoro Timer

Una aplicación de técnica Pomodoro construida con React Native.

## Características

- ⏱️ Temporizador configurable (25 min trabajo, 5 min descanso)
- 📊 Seguimiento de sesiones completadas
- 🔔 Notificaciones al finalizar cada ciclo
- 🎨 Interfaz simple e intuitiva
- 💾 Persistencia de datos

## Instalación

```bash
npm install
# o
yarn install
```

## Uso

```bash
npm start
# o
yarn start
```

## Estructura del Proyecto

```
src/
├── components/
│   ├── Timer.js
│   └── SessionCounter.js
├── screens/
│   └── HomeScreen.js
└── App.js
```

## Tecnologías

- React Native
- React Hooks
- AsyncStorage (para persistencia)
